# RecyclerView万能适配器

##功能
- 支持加载更多
- 支持不同item的添加
- 支持添加头部
- 支持线性与瀑布流显示的切换

##Download
使用Gradle:

```gradle
repositories {
  maven{ url : "https://dl.bintray.com/hanshaofengs/maven" }
}

dependencies {
    compile 'com.hanshao:universallibrary:1.0.0'
}
```
##使用
###创建UniversalAdapter
```
  mUniversalAdapter = new UniversalAdapter();
  
```
